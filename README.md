## Getting started

There are two ways to run the server

```shell
    $ mvn clean spring-boot:run
```

or using docker-compose

```shell
    $ docker-compose run --rm server
```

likewise for execute the tests

```shell
    $ mvn clean test
```

and

```shell
    $ docker-compose run --rm test
```
